# Veterinaria "La Casa del Perro" — Proyecto Final

Bienvenido al repositorio oficial de La Casa del Perro. Este es un sistema de gestión veterinaria modular diseñado para optimizar el control de inventarios, la organización de citas y la autenticación de usuarios.

---

## Características del Sistema

El proyecto está estructurado de forma modular para resolver las necesidades operativas de una clínica veterinaria:

* **Autenticación de Usuarios:** Control de acceso seguro y flujo de inicio de sesión.
* **Gestión de Citas con API de WhatsApp:** Agendamiento conectado con notificaciones directas a través de WhatsApp.
* **Control de Inventario PEPS:** Lógica avanzada de inventariado bajo el método de Primeras Entradas, Primeras Salidas.
* **Panel de Estadísticas Interactivo:** Gráficas visuales y dinámicas desarrolladas con Recharts para el análisis de datos.
* **Persistencia de Datos:** Implementación de almacenamiento local robusto mediante IndexedDB.

---

## Tecnologías Utilizadas

Para el desarrollo de esta aplicación se seleccionó un stack tecnológico eficiente:

* **Frontend:** React (Vite)
* **Estilos:** PersephoneCSS (Diseño responsivo y personalizado)
* **Gráficas:** Recharts
* **Base de datos local:** IndexedDB
* **Control de Versiones:** Git y GitHub

---

## Cómo Visualizar el Proyecto Localmente

Si desea clonar este repositorio para examinar su funcionamiento en un entorno local, ejecute los siguientes comandos en su terminal:

```bash
git clone https://github.com/AlexandraRA23033/Veterinaria_LaCasaDelPerro.git
cd Veterinaria_LaCasaDelPerro
npm install
npm run dev
```
